# vivoh-media-relay

## Usage

`vivoh-media-relay`

## Required components

Needs VLC (or Ffmpeg) installed.

## Command line options

```
-p port of local web server ... eg "8888"
-i IP address to bind to.
-d temporary directory for TS files
-v use VLC as segmenter (default)
-f use Ffmpeg as segmenter
```
